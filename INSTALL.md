# Instalación y Ejecución – ESAPP Frontend

## Requisitos Previos

| Componente | Versión mínima |
|----------|----------------|
| NodeJS   | 22.x           |
| NPM      | 11.x            |

---

## Clonación del Proyecto

```bash
git clone https://github.com/csullaez/esapp-frontend.git
cd esapp-frontend
```
---

## Instalación de Dependencias

```bash
npm ci
```

---

## Ejecución en Desarrollo

```bash
npm run dev
```

Disponible en:
http://localhost:5173

---

## Build de Producción

```bash
npm run build
npm run preview
```

---

## Pruebas

### Unitarias
```bash
npm run test:run
```

### E2E
```bash
npm run test:e2e
```

---

## Linting

```bash
npm run lint
```
